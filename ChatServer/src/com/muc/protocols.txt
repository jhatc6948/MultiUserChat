User & Server

1. User --> Server

    *login / logoff
    *status

2. Server --> User

    * online / offline

3. User --> User

    *direct messages
    *broadcast messages

Commands:

    login <user> <password>
    logoff

    msg <user> body....
    josh: "msg perry hello world"
    perry: "msg josh hello world"

    #topic <-- chatroom / group chat
    join #topic
    msg #topic body...
